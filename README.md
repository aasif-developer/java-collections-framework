# java-collections-framework

## This repository contains my Java Collections Framework learning journey. It includes concepts, examples, and practice programs for all major collection classes in Java.

The Java Collections Framework is a set of interfaces and classes used to store, organize, and manipulate groups of objects efficiently.

## Main Types

- **List**
  - ArrayList
  - LinkedList
  - Vector
  - Stack

- **Queue**
  - PriorityQueue
  - ArrayDeque

- **Set**
  - HashSet
  - LinkedHashSet
  - TreeSet

- **Map**
  - HashMap
  - LinkedHashMap
  - TreeMap
